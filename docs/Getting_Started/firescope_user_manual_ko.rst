FireScope 사용 설명서
======================

개요
----

FireScope는 지도 위에서 항공기, 함정, 기지, 지상 무기체계, 참조점, 임무를 배치하고
시뮬레이션 흐름을 확인하는 작전 편집 도구입니다.

첫 화면
-------

앱은 처음 실행하면 빈 시나리오로 시작합니다.

- 처음에는 배치된 자산이 없는 지도만 보입니다.
- 기존 데모나 저장된 시나리오는 좌측 패널의 ``불러오기``를 눌러야만 표시됩니다.
- 새 배치를 바로 시작하려면 세력을 확인한 뒤 자산을 선택해 지도에 놓으면 됩니다.

빠른 시작
---------

1. 상단의 세력 선택창에서 현재 배치할 세력을 확인합니다.
2. 필요하면 세력 편집에서 세력 이름, 아군/적군 관계, 교리를 조정합니다.
3. 상단의 ``자산 종류`` 버튼 또는 자산 아이콘을 눌러 배치할 종류를 선택합니다.
4. 지도에서 원하는 위치를 클릭해 항공기, 기지, 함정, 지상 무기체계, 참조점을 배치합니다.
5. 배치가 끝나면 좌측 패널의 재생 제어로 1단계 진행, 실행, 일시정지, 속도 변경을 수행합니다.

시나리오 관리
-------------

- ``새로 만들기``: 현재 편집 중인 내용을 비우고 새 빈 시나리오로 다시 시작합니다.
- ``불러오기``: 데모 시나리오, 업로드한 JSON 파일, 클라우드 저장 시나리오를 엽니다.
- ``파일 저장``: 현재 시나리오를 JSON 파일로 내보냅니다.
- ``클라우드 저장``: 로그인 상태에서 현재 시나리오를 서버에 저장합니다.
- ``이름 바꾸기``: 시나리오 제목을 수정합니다.

자산 배치
---------

항공기, 기지, 함정, 지상 무기체계, 참조점은 모두 현재 선택된 세력 기준으로 배치됩니다.

- 항공기: 임무 지정, 무장 편집, RTB, 수동 공격이 가능합니다.
- 기지: 항공기를 보관하고 출격시킬 수 있습니다.
- 함정: 무장 운용과 항공기 탑재가 가능합니다.
- 지상 무기체계: 방공, 화력, 기갑 자산을 지도에 둘 수 있습니다.
- 참조점: 순찰 구역, 임무 경계, 목표 지역 설정에 사용합니다.

임무와 실행
-----------

- ``배치 목록``에서 현재 올린 자산을 확인할 수 있습니다.
- ``집중포격`` 섹션에서는 목표 지정과 전개 상태를 확인할 수 있습니다.
- 항공기와 참조점을 준비한 뒤 순찰 임무 또는 타격 임무를 생성할 수 있습니다.
- ``1단계``는 한 번만 진행하고, ``실행``은 연속 진행, ``일시정지``는 중단입니다.

기록과 재생
-----------

- ``기록`` 섹션에서 시뮬레이션 기록을 저장할 수 있습니다.
- 저장된 기록은 다시 불러와 단계별로 재생할 수 있습니다.
- 리플레이를 열면 현재 단계 기준으로 지도와 로그가 갱신됩니다.

권장 작업 순서
--------------

1. 새 시나리오 또는 불러오기로 시작점을 정합니다.
2. 세력과 관계를 확인합니다.
3. 자산을 지도에 배치합니다.
4. 필요하면 참조점과 임무를 만듭니다.
5. 실행과 단계 진행으로 결과를 확인합니다.
6. 파일 저장 또는 클라우드 저장으로 결과를 보관합니다.
